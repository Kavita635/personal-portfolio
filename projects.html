PROJECTS

1. Student Score Analyzer

A C-based program designed to analyze student academic performance by calculating total marks, averages, grades, and class statistics.

Key Features:

  • Accepts multiple student records
  • Calculates total and average marks
  • Assigns grades based on performance
  • Displays individual student reports
  • Generates class-level statistics

Technologies Used: C, Structures, Functions

Learning Outcome: Improved understanding of data handling and academic performance analysis using C.

2. Student Database Management System

A menu-driven C application that manages student records using file handling. It supports persistent data storage and CRUD operations.

Key Features:

  • Add new student records
  • Display all student data
  • Search student by roll number
  • Update student information
  • Delete student records

Technologies Used: C, File Handling, Structures

Learning Outcome: Learned file handling, data persistence, and building scalable console applications.
