# Personal Portfolio Website

This repository contains a simple personal portfolio website created using pure HTML.

---

## About

The website is a basic multi-page portfolio that displays personal information, academic background, projects, and contact details.  
All pages are connected using hyperlinks and built without CSS or JavaScript.

---

## Project Structure

personal-portfolio  
│  
├── index.html  
├── education.html  
├── projects.html  
├── contact.html  
└── certificates/  

---

## Pages Included

- Home page with basic introduction and navigation  
- Education page with academic details and certifications  
- Projects page with academic projects  
- Contact page with personal contact information  

---

## How to Run

1. Download or clone the repository  
2. Open `index.html` in a web browser  

No setup required.
